# Visualización de datos narrativa

Colección de infografías narrativas y ensayos gráficos publicada como sitio estático bilingüe, lista para compartirse
desde un único repositorio en GitHub.

## Estructura pública

- `index.html`
  Selector de idioma neutral. Es la puerta técnica del repositorio.
- `es/index.html`
  Raíz canónica en español. La home real del proyecto: `Soberanía noética`.
- `en/index.html`
  Canonical English root. The actual home of the project: `Noetic sovereignty`.
- `es/constelaciones/index.html`
  Archivo satélite en español con las piezas previas.
- `en/constellations/index.html`
  English satellite archive with the previous pieces.

## Lógica de navegación

- La última pieza, `Soberanía noética`, pasa a ser la raíz conceptual del proyecto en ambos idiomas.
- Las piezas anteriores se presentan como `constelaciones de ideas`, no como índice principal.
- Cada recurso HTML ofrece alternancia de idioma mediante enlaces estáticos a su versión paralela.
- No hay sesiones, cookies, build ni dependencias: todo está pensado para GitHub Pages o cualquier hosting estático.

## Disclaimer de constelaciones

Las constelaciones incluyen un aviso explícito: mapear un grafo multidimensional completo en web sería posible, pero
de momento no es necesario. Por eso la colección usa una solución más sobria y navegable: piezas satélite enlazadas
entre sí y reconducidas a la raíz.

## Nota práctica

Si publicas este directorio como raíz de un repositorio estático:

- comparte `/es/` para la raíz española
- comparte `/en/` para la raíz inglesa
- comparte `/` si quieres que la persona elija idioma al entrar

El archivo legado `soberania-noetica/soberania-noetica-sadhana.html` se conserva como acceso directo adicional a la
pieza raíz en español, pero la entrada pública recomendada es `es/index.html`.
