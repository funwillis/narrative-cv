# Especialización vs interdisciplinariedad

Infografía narrativa sobre el contraste entre formación especializada e interdisciplinaria, usando el concepto de `banausos` como crítica al estrechamiento funcional y conectándolo con el modelo pedagógico de Jiddu Krishnamurti.

## Archivo principal

- `aprendizaje-especializado-vs-interdisciplinario.html`

## Idea central

La especialización puede producir competencia local muy alta, pero si se absolutiza termina creando sujetos técnicamente útiles y humanamente fragmentados. La salida propuesta aquí no es abolir la profundidad, sino someterla a una educación más amplia: observación, contexto, autoconocimiento y capacidad de conectar campos.
